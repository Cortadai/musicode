# T01: Global media key support

Register globalShortcut or use Electron's built-in MediaSession for media keys (play/pause, next, previous). Send IPC to renderer which dispatches to the audio player.

## Inputs

- `Electron media key APIs`

## Expected Output

- `Media keys control playback in Electron window`

## Verification

Press media keys on keyboard — playback responds
